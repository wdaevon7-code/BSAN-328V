# BSAN-328V

This workspace contains a simple real estate web application built with accessible HTML and responsive CSS.

Files included:

- `index.html` - Home page with navigation, featured image link, and application overview.
- `browse.html` - Browse properties page with a responsive table and links to the liked page.
- `liked.html` - Liked properties page that displays a static example of a saved property.
- `contact.html` - Contact page with an accessible form using fieldsets and labels.
- `style.css` - Responsive design styles for all pages.
