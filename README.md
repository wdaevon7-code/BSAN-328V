# BSAN-328V

This workspace contains a simple real estate web application built with accessible HTML, responsive CSS, and light interactivity.

Files included:

- `index.html` - Home page with navigation, featured image link, and application overview.
- `browse.html` - Browse properties page with a responsive table, like/dislike buttons, and a clickable resource image.
- `liked.html` - Liked properties page that displays properties saved by the user.
- `contact.html` - Contact page with an accessible form using fieldsets and labels.
- `style.css` - Responsive design styles for all pages.
- `script.js` - LocalStorage-based like/dislike support and liked property rendering.
